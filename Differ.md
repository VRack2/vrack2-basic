
Устройство `Differ` (Дифференциатор) предназначено для обнаружения изменений в значениях.


 Оно получает значения на вход `unit` и идентифицирует их по каналу, указанному на входе `channel`. Устройство хранит последнее известное значение для каждого канала. При получении сигнала на входе `gate` (или автоматически, если `gate` не подключен), устройство сравнивает новое значение с сохраненным. Если значение изменилось, устройство отправляет новое значение на выход `unit`, соответствующий идентификатор канала на выход `channel` и сигнал на выход `gate`. 
 

Это позволяет эффективно отфильтровывать дублирующиеся данные и реагировать только на изменения.
