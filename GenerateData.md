Устройство `GenerateData` предназначено для генерации последовательности данных с заданными интервалами. 

Пользователь задает массив объектов, где каждый объект содержит `value` (значение для отправки) и `timeout` (задержка в миллисекундах перед отправкой следующего значения). 

Устройство может быть запущено вручную через вход `start`, остановлено через `stop`, или сброшено к началу массива через `reset`. При достижении конца массива устройство может либо автоматически сброситься и начать цикл заново (`doOnEnd: 'reset'`), либо остановиться (`doOnEnd: 'nothing'`). Состояние запуска и текущий индекс отслеживаются через `shares`.
