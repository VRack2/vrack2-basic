Микшер предназначен для суммирования нескольких входов в один или несколько специализированны групп выходов. 

Основные входы (`mix1...`) могут принимать любое значение, после чего это значение будет перенаправлено на комбинированную группу выходов. Комбинированная группа состоит из 3х выходов `mixed`, `channel` и `gate`

Данный микшер поддерживает комбинированные группы входов и выходов `FX` которые позволяют перенаправить любой основной вход (`mix1...`) на комбинированную группу выходов `FX`, а так же принять данные на комбинированную группу выходов `FX` и перенаправить их в основной комбинированный выход. 

Для настройки маршрутизации можно использовать параметром `routes`

```
"routes": {
    "Номер основного входа": "название выхода"
}
```

Пример отправки `mix2` в комбинированную группу выходов `fx1`

```json
{
    "id": "Mixer1",
    "type": "basic.Mixer",
    "params": {
        "inputs": 8,
        "outputsFX": 2, 
        "routes": {
            "2": "fx1"
        }
    }
}
```

Для назначения сразу нескольких групп можно перечислить их в массиве

```json
"routes": {
    "2": ["main","fx1","fx2"]
}
```

Если основной вход участвует в маршрутизации, необходимо явно указывать в качестве маршрута `main` для отправки данны в основную комбинированную группу выходов. Если вход не участвует в маршруте, то он всегда будет отправлятся только в основную комбинированную группу выходов.
